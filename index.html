Deep Research

Multi-step autonomous research agent. Plans sub-questions, searches the web, reads sources, synthesizes findings into structured reports with citations.

⚡

Real-time Streaming

Server-Sent Events stream every step as the agent works. Watch it plan, search, read, analyze, and synthesize in real time. Or poll — your choice.

📋

Fact Checking

Submit any claim. The agent searches for evidence, evaluates credibility, and returns a verdict with confidence score and supporting/contradicting evidence.

📈

Entity Comparison

Compare 2+ entities across any dimensions. Products, companies, frameworks, cities — get a structured comparison table with scores and recommendations.

🌐

Data Extraction

Point it at any URL. The agent reads the page and extracts structured data — entities, facts, dates, numbers. Bring your own schema or let it auto-detect.

🔒

Multi-LLM Backend

Anthropic Claude, OpenAI GPT, Google Gemini. Different models for planning vs. research vs. synthesis. Automatic failover. Zero vendor lock-in.

API Reference

POST /v1/research Submit deep research query — returns task ID for polling ASYNC
POST /v1/research/quick Quick research — synchronous, returns report immediately SYNC
GET /v1/research/:id Poll task status and retrieve results
GET /v1/research/:id/stream SSE stream — real-time step-by-step progress SSE
POST /v1/fact-check Verify claims against evidence — verdict + confidence SYNC
POST /v1/compare Compare entities — structured table + recommendation SYNC
POST /v1/extract Extract structured data from any URL SYNC
GET /health Health check — providers, search status, active tasks
